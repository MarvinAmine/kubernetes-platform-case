# Reliability Dashboards

This folder stores service-specific Grafana dashboard JSON files owned by the
Reliability Team.

## Ownership

- Platform team owns the shared Grafana stack, provider wiring, Helm
  integration, and shared platform dashboards.
- Reliability Team owns the service-health dashboard content in this folder.

## Expected content

Use this folder for service-specific operational dashboards, for example:

- request and error rate
- latency views
- connection pool health
- application-specific operational signals
